# Email/SMS Spam Filter Project

This project classifies messages as spam or not spam using machine learning (Python, scikit-learn, pandas).

## Overview

A classic text classification project using the [SMS Spam Collection Dataset](https://archive.ics.uci.edu/ml/datasets/SMS+Spam+Collection).  
It demonstrates message preprocessing, vectorization, training, evaluation, and prediction using Naive Bayes.

## Features

- Loads and explores SMS data.
- Preprocesses messages (TF-IDF vectorization, stop word removal).
- Trains a multinomial Naive Bayes spam classifier.
- Evaluates accuracy and classification report.
- Predicts spam/ham for new messages.
